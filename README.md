# CRM

#### 介绍
客户关系管理系统，韬睿tr1807班结业项目

#### 目录结构

- back_end：后端工程
- data_base：数据库脚本
- demand：需求、开发文档
- front_end：前端工程

#### 软件架构
SpringBoot、MySQL、VUE、WebStrom、IDEA


#### 安装教程

- 后端直接克隆即可

  ```bash
  git clone https://gitee.com/baixuanzi/crm.git
  ```

- 前端

  克隆下来后，前端工程（front_end）一并在文件夹（crm）里；

  进入front_end，命令行安装依赖即可

  ```bash
  cnpm install
  ```
